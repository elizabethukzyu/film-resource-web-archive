# film-resource-web-archive
本仓库主要用于记录影视相关网站的基础资料信息，适合用于长期信息备忘与归档管理。整理内容包括站点名称、访问地址及站点类型等公开字段，所有信息均采集自公开渠道。仓库不存储影视内容，不提供播放或下载功能，仅作为站点信息整理工具存在，不代表与所记录网站存在任何合作或商业关系。
